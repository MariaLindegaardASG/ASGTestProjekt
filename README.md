The project is built with the following technologies:

- **SASS**: CSS pre-processor
- **Tailwind CSS**: Utility-first CSS framework
- **ESLint**: Linter for ensuring code quality
- **Prettier**: Code formatting tool

## **Installation**

1. **Clone this repository:**

   ```bash
   git clone https://github.com/your-username/car-lease-portal.git
   cd car-lease-portal
   ```

2. **Install necessary packages:**
   Ensure that Node.js and npm are installed on your machine. Then install the required npm packages:

   ```bash
   npm install
   ```

3. **Tailwind CSS Setup:**
   Tailwind CSS is already configured in the project. You can find the configuration in `tailwind.config.js` and `src/tailwind.scss`.

4. **Configure ESLint and Prettier:**
   ESLint and Prettier are installed and configured to work together, ensuring consistent code style and high code quality.

## **Setup**

1. **Environment Variables:**
   If the application requires environment-specific variables, they can be configured in Angular's environment files (`src/environments/`).

2. **Backend API URL:**
   Ensure the API URL in `car.service.ts` is updated to match the correct backend URL.

## **Running the Application**

To run the application locally:

1. **Start the development server:**

   ```bash
   ng serve
   ```

2. **Open your browser and navigate to:**
   ```
   http://localhost:4200/
   ```

The application will now be accessible on your local development server.
